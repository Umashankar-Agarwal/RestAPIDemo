This repository contains an automated testing framework for REST APIs, developed as part of a Udemy learning course. The project demonstrates industry-standard practices for API validation, request structuring, and test reporting.

🚀 Key FeaturesFull 
CRUD Validation: Tests GET, POST, PUT, and DELETE endpoints.
Data-Driven Testing: Separates test data from execution logic using external files.
Schema Validation: Ensures API responses match expected JSON schemas.
Automated Reporting: Generates detailed HTML reports for test runs.
CI/CD Ready: Configured for command-line execution and pipeline integration

.🛠️ Tech Stack
Language: Java
Testing Framework: REST Assured 
Build Tool: Maven 
Reporting: Allure Reports / Extent Reports / HTML 

Extra📋 PrerequisitesBefore running the project, ensure you have the following installed:
SDK/Runtime: Java JDK 11+ / Python 3.8+ / Node.js
Build Tool: Maven
Editor: IntelliJ IDEA / VS Code / PyCharm
